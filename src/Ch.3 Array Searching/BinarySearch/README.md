# 3-1 線性搜尋

- 何謂線性搜尋
    - 從最前端一直往後尋找直到最後端
    - 途中如果找到就返回
    - 不需要事前整理資料結構
    - 時間複雜度為**O(n)**

- example
    - Javascript中提供的array methods: find, findIndex etc...
