# 2-6 合併排序

- 何謂合併排序
    - 拆分步驟
        - 把大陣列切一半成為兩個小陣列
        - 重複上步驟直到每個小陣列都只剩一個元素
    - 合併步驟
        - 把兩邊排序好的小陣列合併並排序成一個陣列
        - 重複上步驟直到所有小陣列都合併成一個大陣列
    - 執行時間 O(n log n)
