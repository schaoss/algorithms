# 2-3 選擇排序

- 何謂選擇排序
    - 搜尋數列中的最小值，將它與最左邊的數對調
    - 搜尋最小值使用線性搜尋（可參考 p.082）
    - 比較的總次數：(n-1) + (n-2) + ...... + 1 ≈ n²/2
    - 執行時間 O(n²)
