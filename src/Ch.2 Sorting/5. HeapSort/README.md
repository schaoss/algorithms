# 2-5 堆積排序

- 何謂堆積排序
    - 先整理出堆積結構
    - 將根取出放到數列最右邊
    - 重新排列堆積
    - 重複執行以上步驟直到堆積清空
    
    - 執行時間 O(n log n)
