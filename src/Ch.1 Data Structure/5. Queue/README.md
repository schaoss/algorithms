# 1-5 佇列

## 重點摘錄
- 何謂佇列
    - 先進先出 First In First Out (FIFO)

- 特性
    - 追加和刪除數據在相反邊進行
    - 如需中間數據，需先輸出佇列前面所有的數據
    - 插入、刪除需時 O(1) 搜尋需時 O(N) 


- 例子
    - 廣度優先搜尋（p.092）
